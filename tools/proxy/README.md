# proxy — DigitalOcean nginx reverse proxy

Provisions a single DigitalOcean droplet running **nginx** that terminates TLS
on `:443` and forwards all traffic to an upstream service on `:3000`, fronted by
a **proxied Cloudflare** DNS record.

## What it creates

| Resource                        | Notes                                                       |
| ------------------------------- | ----------------------------------------------------------- |
| `digitalocean_droplet.proxy`    | nginx, provisioned via [cloud-init/nginx.yaml.tftpl](cloud-init/nginx.yaml.tftpl) |
| `digitalocean_firewall.main`    | inbound `:22` + `:443`, all egress                          |
| `digitalocean_project.main`     | groups the droplet                                          |
| `cloudflare_record.proxy`       | **proxied** `A` record `<subdomain>.<parent_domain>` → droplet IP |

## Request flow

```
client ──TLS──▶ Cloudflare edge ──TLS(:443)──▶ nginx droplet ──HTTP(:3000)──▶ upstream
                (public cert)      (self-signed origin cert)
```

- Cloudflare proxies the public hostname (orange cloud) and terminates the
  public TLS connection.
- nginx serves a **self-signed** origin certificate generated by cloud-init, so
  set the Cloudflare **SSL/TLS mode to `Full`** for this zone. For `Full
  (strict)`, swap in a [Cloudflare Origin CA](https://developers.cloudflare.com/ssl/origin-configuration/origin-ca/)
  certificate.
- nginx `proxy_pass`es to `var.upstream_host:var.upstream_port` (default `:3000`),
  with WebSocket upgrade headers for Semaphore's live task-log streaming.

## Setup

```sh
cd tools/proxy

# SSH key the droplet trusts (workspace = "denis" -> keys/proxy-denis/):
mkdir -p ../../keys/proxy-$(terraform workspace show)
ssh-keygen -t rsa -b 4096 -N '' -f ../../keys/proxy-$(terraform workspace show)/id_rsa

cp terraform.tfvars.example terraform.tfvars   # fill in tokens + upstream_host
terraform init
terraform apply
```

Required variables (see [variables.tf](variables.tf)):

- `do_token`, `cloudflare_api_token` — API tokens.
- `parent_domain` — Cloudflare-hosted parent zone (e.g. `semaphoreui.dev`).
- `upstream_host` — backend the proxy forwards `:443` traffic to (its `:3000`).
- `subdomain` *(optional)* — record label; defaults to `proxy-<workspace>`.

After apply, the public URL is printed as `proxy_url`
(`https://<subdomain>.<parent_domain>`).
